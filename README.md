# Student Vault Pro

A modern student file management platform built for hackathons.

## Overview

Student Vault Pro helps students securely manage and organize their academic files in one place.

## Features

- User Login and Registration
- Student Dashboard
- Upload Files
- View My Files
- Search and Categories
- User Profile

## Tech Stack

Frontend:
- React + Vite
- React Router
- CSS

Backend:
- Firebase (Integration in progress)

## Project Status

Frontend development in progress.
